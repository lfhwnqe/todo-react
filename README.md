# todolist
### 组件
  1. todobox
    - 包含所有组件
  2. todoitem
    - 单条信息
    - 点击勾选功能，勾选划线，与todostatus组件互动，状态组件会显示未被勾选的信息
    - 焦点显示删除按钮，点击删除，删除也会影响状态组件。
  3. todoinput
    - 输入信息
    - 向下箭头点击勾选全部按钮。
    - 回车键确定输入信息
  4. todostatus
    - 信息状态，完成与未完成
    - 选中状态更改判断条件，根据判断显示选中类型
    - 如果有完成状态的数据，显示清除按钮，点击可以清除
  5. todolist
    - 包含所有todoitem
